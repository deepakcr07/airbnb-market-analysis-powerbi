# Airbnb Market Analysis Dashboard

## Project Overview
This project presents an interactive Power BI dashboard designed to analyze Airbnb market trends, pricing patterns, room availability, host activity, and review performance.

## Business Problem
Airbnb listings are spread across different boroughs, room types, and price ranges. This dashboard helps identify market patterns and supports better decision-making.

## Dashboard Preview
![Dashboard](Screenshots/dashboard.png)

## Key Metrics
- Average Pricing: 152.76
- Total Listings: 49K
- Total Hosts: 37K
- Average Availability: 112.78
- Reviewed Listings: 0.79

## Features
- Interactive filters for neighbourhood group, price category, review status, room type, and availability status
- Airbnb map visual
- Average price by borough
- Reviews over time
- Listings by room type
- Price vs availability analysis

## Tools Used
- Power BI
- Power Query
- DAX
- Data Visualization
- Data Cleaning

## Key Insights
- Manhattan shows the highest average price.
- Entire home/apartment is the most common room type.
- Listings are concentrated in major boroughs.
- Availability and pricing show a noticeable relationship.
